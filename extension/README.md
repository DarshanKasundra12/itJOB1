# IT Company Finder - Chrome Extension

This extension finds IT companies near any location using 100% free APIs (OpenStreetMap Overpass & Nominatim).

## Installation Instructions

1. **Download** the files in this folder to your computer.
2. Open Google Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** in the top right corner.
4. Click **Load unpacked**.
5. Select the folder where you saved these files.
6. The extension icon will appear in your browser toolbar.

## Features

- **Automatic Location Detection**: Uses your IP to find your city.
- **Search**: Type any place name (e.g., "Vastral", "Maninagar").
- **Map Visualization**: Shows your location (Blue) and IT companies (Red).
- **Smart Contact Info**: Auto-generates probable HR emails based on company names/websites.
